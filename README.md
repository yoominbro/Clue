# clue
1인용 클루 룰 

	1.	정답은 용의자 1명 + 무기 1개 + 장소 1곳이며 이것을 모두 맞추는 것이 승리 조건이다

	2.	나머지 카드(용의자/무기/장소)를 모두 섞어 4개를 제공 받음

	3.	너는 가지고 있는 카드들은 정답에서 제외되는 정보
  
	4.	보드판은 랜덤 이동 or 무시하고, 턴마다 임의의 장소를 방문했다고 가정 가능하며 방문한 장소로만 추리
  
	5.	장소에 갈 때마다 “용의자 + 무기 + 그 장소”로 **가설(의심 조합)**을 하나 만들기
  
	6.	가설을 만들면, 제공받은 패를 보고 배제할 수 있는 카드가 있는지 체크
  
	7.	배제할 카드가 없다면, 그 조합은 정답일 가능성 상승(각자 종이에 메모)
  
	8.	게임은 이런 식으로 모든 조합을 탐색하며 후보를 줄여가는 방식으로 진행
  
	9.	최종적으로 정답 후보가 하나로 남았을 때 최종추리를 진행한다

	10.	맞으면 승리! 


# 설치방법 
1. 파일 다운로드 zip으로 받기
2. 터미널에서 pip install pillow 로 pillow 라이브러리 설치
3. clue_gui.py 실행 후 run
4. 게임 실행
